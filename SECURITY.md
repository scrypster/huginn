# Security Policy

## Reporting a Vulnerability

Please **do not** report security vulnerabilities through public GitHub issues.

Instead, open a [GitHub Security Advisory](https://github.com/scrypster/huginn/security/advisories/new) with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fix (optional)

We'll keep you updated as we work through the fix and will credit you in the release notes unless you prefer to remain anonymous.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest release | ✅ Full support |
| Previous release | ✅ Critical patches only |
| Older | ❌ No support |

## Scope

In scope: the Huginn binary, server, relay, MCP client, web UI, and any bundled integrations.

Out of scope: third-party MCP servers, Ollama, or external APIs that Huginn connects to.
